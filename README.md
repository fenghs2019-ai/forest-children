# 爱（AI）解题叶渡诗河

**森林的孩子系列 · 故事诗结构化分析平台**

---

## 平台概述

这是一个专用于分析《森林的孩子》故事诗系列的结构化平台，包含162首以单字为题的故事诗。平台提供完整的AI结构化分析，包括：

- 原文呈现
- 故事概述
- 故事背景
- 人物简介与关系图谱
- 故事时间线与重大事件
- 故事价值观
- 章节题目与关键内容标签

---

## 如何使用

### 1. 浏览诗河

打开 `index.html`，可以看到：

- **统计面板**：总诗数、已拆解数、待处理数、完成率
- **诗河导航**：162个单字以圆形按钮呈现
  - 绿色按钮：已拆解，可点击进入详情页
  - 灰色按钮：待处理

### 2. 查看诗歌分析

点击任何绿色按钮（已拆解的诗歌），会跳转到详情页：

- 顶部有"返回诗河"按钮
- 7个标签页：原文、概述、背景、人物、时间线、价值观、章节
- 所有数据从对应的JSON文件动态加载

### 3. 添加新诗歌的拆解

当你有了新的诗歌拆解后（通过AI分析工具生成），执行以下步骤：

1. 在根目录创建 `output-字/` 文件夹（例如 `output-念/`）
2. 将以下6个文件放入该文件夹：
   - `analysis.json`
   - `analysis.md`
   - `analysis.txt`
   - `analysis.xml`
   - `index.html`
   - `report.html`
3. 更新 `data/index.json`：
   ```json
   {
     "id": "04",
     "char": "念",
     "status": "completed",
     "path": "output-念/index.html"
   }
   ```
4. 更新统计数据：
   ```json
   {
     "total": 162,
     "completed": 4,
     "pending": 158,
     "completion_rate": "2.47%"
   }
   ```
